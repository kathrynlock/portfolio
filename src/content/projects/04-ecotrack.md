HCI course project built in 3 weeks. The challenge: make sustainability feel rewarding instead of guilt-inducing. Core mechanic is a virtual plant that grows as you log eco-friendly habits — it wilts if you miss a day but recovers quickly, which was a deliberate choice to avoid shame spirals.

Prototyped in Figma, coded in React Native. User tested with 12 students. Top-rated project in the course. If I had another semester I'd add a community layer — the plant mechanic works better with an audience.
